# whatdojuniorsgetpaid

Anonymously share your salary stories, and help others to learn from your experiences.

ruby version 2.2.2

rails version 4.2.3

### Get the code

Get the code from this repo

    git clone

### Run locally

Install all the gems

    bundle install

Spice up the database

    rake db:migrate

Get some data

    rake db:seed

Run the server

    rails s
